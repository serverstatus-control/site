• ⚪STATO NON INDICATO: impossibile sapere lo stato dei server
  • 🟤SERVER IN MANUTENZIONE PROGRAMMATA: i server sono chiusi oppure sono inutilizzabili per manutenzione programmata
  • 🔵SERVER PARZIALMENTE CHIUSI PER BASSA AFFLUENZA: alcuni server sono chiusi vista la bassa affluenza di utenti
  • 🟠SERVER CON ALTA AFFLUENZA: i server potrebbero essere rallentati (rischiando il crash) per alta affluenza di utenti
  • 🟢SERVER OPERATIVI: tutto funziona perfettamente
  • 🟡🟢SERVER CON POSSIBILI PROBLEMI: potrebbero esserci problemi con i server o problemi molto lievi
  • 🟢🟡SERVER IN GRADUALE RIPRESA: server in ripresa dopo un down
  • 🟡SERVER CON PROBLEMI: potrebbero esserci dei malfunzionamenti sul sito/gioco/app/ecc...
  • 🟠SERVER OPERATIVI CON PROBLEMI GRAVI: la maggior parte delle funzioni non è disponibile o ha dei problemi gravi, alcuni server in down
  • 🟠SERVER IN DOWN PARZIALE: la maggior parte delle funzioni non sono diponibili, alcuni server meno importanti in down
  • 🔴SERVER IN DOWN: server non funzionanti, molte funzioni non sono disponibili (social, siti, pagamenti, ecc...)
  • 🟣TUTTI I SERVER IN DOWN: tutte le piattaforme e le funzionalità non sono disponibili (siti, giochi, app, social...)
  • 🟣ATTACCO HACKER IN CORSO: tutte le piattaforme e le funzionalità non sono disponibili (siti, giochi, app, social...) a causa di un attacco hacker
  • ⚫SERVER DISTRUTTI O CHIUSI PER SEMPRE: i server sono per sempre inutilizzabili per incidenti o chiusure programmate e non potranno essere riattivati per fallimento
